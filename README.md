## 1. Requirements

- Node.js >= v8.11.3
- estore-backend: <https://github.com/vergil465/estore-backend/>

## 2. Installation

1. Run npm command in package directory `npm install`.
2. After installation run `npm run start-dev`.

## 3. URLs

- LOCAL_API - <http://localhost:3000>
- LIVE_API - <https://estore-app.herokuapp.com>
